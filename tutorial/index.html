Play & Spark

Apache Spark is a fast and general engine for large-scale data processing. This Typesafe Activator template demonstrates Apache Spark working within the play framework serving REST request.

The demo app

Some data (a dozen tweets) are read from json file in the resources/tweet-json directory. This could be parquet or whatever format relevant to load spark DataFrame

Run the App

activator ~run
and hit url like http://localhost:9000/count, http://localhost:9000/list or http://localhost:9000/filter/Scala,
